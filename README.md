# Runa's Kits

Runa's core utilities.  Please use and enjoy.

Usage
-----

```clj
[org.clojars.runa/kits "1.19.10"]
```

```clj
(require '[kits.map :refer [map-keys]])
```

Tests
-----

Run tests vs all supported Clojure versions:

```
lein run-tests # using Leningen 2
```
